Esendex's PEAR Channel Server

Registering the channel: 
pear channel-discover esendex.github.com/pear
Listing available packages: 
pear remote-list -c esendex
Installing a package: 
pear install esendex/package_name
Installing a specific version/stability: 
pear install esendex/package_name-1.0.0
pear install esendex/package_name-beta
Receiving updates via a feed: 
http://esendex.github.com/pear/feed.xml

Esendex - Esendex PHP REST SDK

The Esendex PHP SDK for developing SMS capable applications using our REST API. This library enables base functionality and examples of how to interact with the API.
